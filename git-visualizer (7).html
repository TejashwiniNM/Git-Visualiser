👋 Welcome to Git Visualizer
This tool teaches Git by letting you click through every command and watch a live simulated GitHub repository react in real time. Here's how to use it:
1 Click "Next →" to start the guided tour
The bottom-right Next → button selects the next command AND runs it automatically — like a slideshow. Prev ← goes back. This is the easiest way to learn — just keep clicking Next.
2 Or click any command directly
Browse the left panel (organized Beginner → Expert). Click any command to select it, read the info card showing syntax + description + safety tag, then click ▶ Run Command.
3 Watch the right side update like real GitHub
Every command changes the simulated repo. Switch between tabs to see different views:
TabWhat it shows
📁 CodeFile tree — untracked (orange), staged (green), modified, committed
🕐 CommitsVisual branch graph with commit history
🌿 BranchesAll branches, current branch (HEAD), and tags
🔍 DiffLine-by-line changes before staging
🔄 ZonesThe 4 Git zones — see files move between them live
4 Yellow "Try Next" suggestions
After running a command, recommended next commands get highlighted with a 👉 Try Next badge — so you always know what to do next instead of guessing.
5 Terminal output + Git Guide
The bottom-right panel shows real Git terminal output for every command, including real error messages if something's wrong. Click 📖 Git Guide anytime for deeper explanations, tables, and best practices.
💡 Tip: Start from the top of the left panel (git init) and just keep pressing Next → — by the end you'll understand the entire Git + GitHub workflow.
🎯 Challenges
Pick a goal. Use any commands (click cards, Next/Prev, or type in the terminal) to reach that state. The visualizer detects when you've completed it.
🎯 Challenge Mode
⚠️ Starting this challenge resets your simulated repo to a clean slate, so you can solve it from scratch.
💡 Stuck? A "Hint" button appears once the challenge starts.
